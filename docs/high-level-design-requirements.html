Mainnet proof

  • Walrus Site object deployed on Mainnet.
  • 104 fixtures registered in the Sui MatchRegistry.
  • Team profiles and gallery media expose Walrus blob pointers.
  • Daily What's Up exposes blob, memory namespace, Sui receipt, and novelty proof.

Storage split

  • Walrus Memory: global schedule and wallet notebooks.
  • Briefing Memory: editorial memory for anti-repeat writing.
  • Walrus Blob: media, team data, generated payloads.
  • Sui: signed receipts, gate state, content hashes.
  • Supabase: rebuildable cache for fast UI reads.
