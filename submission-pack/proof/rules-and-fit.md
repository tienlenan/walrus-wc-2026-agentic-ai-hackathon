## Judging Criteria

1. Memory Depth and Authenticity
   - Does memory change agent behavior over time?
   - Is there a real before/after moment?

2. Creativity and Flair
   - Is the project fun, unexpected, and shareable?
   - Does it use the World Cup context meaningfully?

3. Technical Execution and Completeness
   - Is it live on Walrus Mainnet?
   - Is Walrus Memory correctly integrated?
   - Does the core experience work?
